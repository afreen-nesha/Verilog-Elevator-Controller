# Verilog Elevator Controller

## Overview

This project implements a **3-floor elevator controller using Verilog HDL**.
The system is designed using a **Finite State Machine (FSM)** to control elevator movement, direction, and door operation based on floor requests.

## Features

* FSM-based control logic
* Handles floor requests
* Direction control (up / down)
* Door opens when the destination floor is reached
* Testbench included for simulation
